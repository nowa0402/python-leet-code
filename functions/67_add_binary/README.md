# Add Binary

<https://leetcode.com/problems/add-binary/description/>

## **2つのバイナリ文字列の和**

2 つの **バイナリ文字列** `a` と `b` が与えられます。  
**2 つの文字列の和** を **バイナリ形式の文字列** として返してください。

---

## **例**

### **例 1**

**入力:**  
`a = "11", b = "1"`  
**出力:**  
`"100"`  

---

### **例 2**

**入力:**  
`a = "1010", b = "1011"`  
**出力:**  
`"10101"`  

---

## **制約**

- `1 <= a.length, b.length <= 10⁴`
- `a` と `b` は **'0' または '1'** だけで構成される。
- 各文字列には **先頭に `0` は含まれない**（`0` 自体を除く）。
